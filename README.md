# 💎 LabelByPooja - Premium E-Commerce Platform

LabelByPooja is a modern, high-end e-commerce application built with React and Vite.

## ✨ Features
- Premium UI with glassmorphism
- Light/Dark theme
- Product filtering & search
- Wishlist & cart
- Animations using Framer Motion
- Responsive design

## 🚀 Run Project
npm install  
npm run dev
